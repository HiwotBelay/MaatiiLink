# Pilot Training — Branch Champions

**Audience:** 1 champion per pilot branch (manager or senior staff)  
**Duration:** 45 minutes + 15 min Q&A  
**Materials:** Staging URL, dev guide `docs/guides/BRANCH_MANAGER_GUIDE.md`

## Agenda

1. **Login & security** (5 min)  
   - Use bank-issued account only; log out on shared PCs.

2. **Digital EOD** (15 min)  
   - Draft → submit before cutoff (Addis time).  
   - Supervisor lock workflow.

3. **HO directives** (10 min)  
   - Read published directive; acknowledge before deadline.

4. **Incidents** (5 min)  
   - Log severity correctly; critical = immediate call to supervisor.

5. **Service desk** (5 min)  
   - Open ticket for IT / facilities; track status.

6. **Pilot feedback** (5 min)  
   - `/pilot` → Submit pilot feedback for bugs, UX, training gaps.  
   - Sev-1 only for production blockers.

## Hands-on checklist (each champion)

- [ ] Log in successfully
- [ ] Submit today’s EOD (or draft if after cutoff)
- [ ] Acknowledge latest directive
- [ ] Submit one test feedback item (LOW severity)

## Support contacts

Document bank-approved channels in your pilot packet (supervisor phone, HO IT desk). Do not put personal phone numbers in the repo.
